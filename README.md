## Descripción

Este proyecto implementa el modelo de dominio de un sistema bancario utilizando principios de programación orientada a objetos y arquitectura DDD (Domain-Driven Design).

El sistema permite representar:

- Clientes (personas y empresas)
- Cuentas bancarias
- Préstamos
- Transferencias
- Usuarios del sistema
- Bitácora de operaciones

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Maven
- Git & GitHub

## Arquitectura

El proyecto sigue una arquitectura por capas:

- Model (Dominio)
- Service
- Repository
- Controller
